# cboettig.r-universe.dev

This repository contains the registry for my [R-universe](https://cboettig.r-universe.dev).

## Packages

| Package | Source |
|---------|--------|
| [roxigraph](https://cboettig.r-universe.dev/roxigraph) | [GitHub](https://github.com/cboettig/roxigraph) |

## Installation

```r
# Enable this universe
options(repos = c(
  cboettig = 'https://cboettig.r-universe.dev',
  CRAN = 'https://cloud.r-project.org'))

# Install packages
install.packages('roxigraph')
```
